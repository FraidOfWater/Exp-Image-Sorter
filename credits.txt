**This project makes use of at least the following libraries and tools (and is under the licenses they allow):**

1. **Pillow**
   - Website: [https://python-pillow.org/](https://python-pillow.org/)
   - Description: Pillow is a Python Imaging Library (PIL) fork that adds image processing capabilities to your Python interpreter. It supports opening, manipulating, and saving many different image file formats.
   - License: PIL License

2. **pyvips**
   - Website: [https://libvips.github.io/pyvips/](https://libvips.github.io/pyvips/)
   - Description: pyvips is a Python binding for the VIPS image processing library, which is designed for high-speed image processing and is particularly well-suited for large images.
   - License: LGPL-2.1

3. **imageio**
   - Website: [https://imageio.readthedocs.io/](https://imageio.readthedocs.io/)
   - Description: imageio is a Python library that provides an easy interface to read and write a wide range of image data, including animated images, volumetric data, and scientific formats.
   - License: BSD License

4. **imageio[ffmpeg]**
   - Website: [https://imageio.readthedocs.io/en/stable/ffmpeg.html](https://imageio.readthedocs.io/en/stable/ffmpeg.html)
   - Description: This extension of imageio allows for reading and writing video files using FFmpeg, a powerful multimedia framework.
   - License: BSD License

5. **NumPy**
   - Website: [https://numpy.org/](https://numpy.org/)
   - Description: NumPy is a fundamental package for scientific computing in Python. It provides support for arrays, matrices, and a large collection of mathematical functions to operate on these data structures.
   - License: BSD License

6. **python-vlc**
   - Website: [https://www.videolan.org/doc/python-vlc/](https://www.videolan.org/doc/python-vlc/)
   - Description: python-vlc is a Python binding for the VLC media player, allowing you to control VLC from Python scripts.
   - License: GPL v2

7. **psutil**
   - Website: [https://psutil.readthedocs.io/](https://psutil.readthedocs.io/)
   - Description: psutil is a cross-platform library for retrieving information on running processes and system utilization (CPU, memory, disks, network, sensors) in Python.
   - License: BSD License

8. **PyInstaller**
   - Website: [https://www.pyinstaller.org/](https://www.pyinstaller.org/)
   - Description: PyInstaller is a program that converts Python applications into stand-alone executables, under Windows, Linux, and Mac OS X.
   - License: GPL v2

9. **Simple-Image-Sorter**
   - Website: [https://github.com/Legendsmith/Simple-Image-Sorter](https://github.com/Legendsmith/Simple-Image-Sorter)
   - Description: Sorts images into destination files. Thanks a bunch for the original program to Legendsmith. Special thanks to FooBar167 from the original author.

10. **VLC Media Player**
    - Website: [https://www.videolan.org/vlc/](https://www.videolan.org/vlc/)
    - Description: VLC Media Player is a free and open-source cross-platform multimedia player and framework that plays most multimedia files, as well as DVDs, audio CDs, VCDs, and various streaming protocols.
    - License: GPL v2

11. **VIPS**
    - Website: [https://www.vips.ecs.soton.ac.uk/](https://www.vips.ecs.soton.ac.uk/)
    - Description: VIPS is a free and open-source image processing system that is designed for high-speed image processing and is particularly well-suited for large images.
    - License: LGPL-2.1
