This project makes use of the following libraries and tools: (And is under whatever licence they allow)

1. VLC Media Player
   - Website: https://www.videolan.org/vlc/
   - Description: VLC is a free and open-source cross-platform multimedia player and framework that plays most multimedia files, as well as DVDs, audio CDs, VCDs, and various streaming protocols.

2. VIPS
   - Website: https://www.vips.ecs.soton.ac.uk/
   - Description: VIPS is a free and open-source image processing system that is designed for high-speed image processing and is particularly well-suited for large images.

3. Simple-Image-Sorter
	- Website: https://github.com/Legendsmith/Simple-Image-Sorter
	- Sorts images into destination files.
	- Thanks a bunch for the original program to Legendsmith. Special thanks to FooBar167 from original author.
